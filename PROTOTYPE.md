# Null Cycle – Prototype

This prototype demonstrates the core mechanics of the project.

Implemented systems:
- Autonomous NPC movement
- Sequential spawning
- Collision and direction change
- Player placed blocks
- Death zones and exit zones
- Signal integrity affecting movement speed

The goal of this prototype was to validate the core gameplay loop of a Lemmings-style puzzle system using AI-assisted development.

This prototype serves as the technical foundation for the main project.
